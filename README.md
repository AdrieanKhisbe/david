David
=====

Nodejs based web service that tells you when your project npm dependencies are out of date.

Getting Started
---------------

Install [Node.js](http://nodejs.org/)

Install grunt:

	npm install -g grunt

Install dependencies:

	npm install request
	npm install npm
	npm install express
	npm install moment

Run david:

	node david
